# Эмпирическая оценка бюджетного правила Норвегии

## Описание
Учебный проект, посвящённый эконометрическому анализу бюджентоно правила Норвегии

## Подход
- Поиск и анализ информации по конфигурации правил
- Анализ объясняющих и зависимой переменных
- Построение регрессионной модели
- Анализ статистической значимости коэффициентов
- Проверка устойчивости модели
- Интерпретация результатов с макроэкономической точки зрения
